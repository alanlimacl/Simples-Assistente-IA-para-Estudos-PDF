### Descrição do Projeto

Este projeto é uma IA simples para análise de documentos em PDF.
O usuário envia um arquivo PDF, e a aplicação lê, interpreta o conteúdo e responde perguntas com base exclusivamente nas informações presentes no documento.

A IA extrai o texto do PDF, processa os dados e gera respostas contextualizadas, tornando mais fácil consultar contratos, relatórios, artigos ou qualquer outro documento sem precisar ler tudo manualmente.

#### Principais funcionalidades:
- Upload de arquivos PDF

- Extração automática do texto do documento

- Análise do conteúdo usando IA

- Respostas baseadas apenas no que está escrito no PDF

- Interface simples e intuitiva

#### Objetivo:
O objetivo do projeto é demonstrar o uso de Inteligência Artificial aplicada à leitura e compreensão de documentos, focando em automação, produtividade e aprendizado prático de IA.
